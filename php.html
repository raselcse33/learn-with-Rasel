PHP Introduction scripting language : স্ক্রিপ্টিং ল্যাঙ্গুয়েজে হচ্ছে এক প্রকার প্রোগ্রামিং ল্যাঙ্গুয়েজে যেটা compilation ছাড়াই excute হয়। example : php ,javascript of Non-Scripting Languages : নন স্ক্রিপ্টিং ল্যাঙ্গুয়েজে হচ্ছে Compiled ল্যাঙ্গুয়েজে। মানে এই ল্যাঙ্গুয়েজে টাকে কম্পিউটার এ রান করার পূর্বে এটি কে compiled অর্থাৎ মেশিন ল্যাঙ্গুয়েজে (বাইনারি কোড এ ) রূপান্তর করতে হবে। example : c ,c ++,java compiled : compilied হলো সেই জায়গা যেখানে আমার লেখা কোড টাকে machine language (binary code) convert করা হয়, যাতে কম্পিউটার বুজতে পারে এবং ঠিক ভাবে রান করতে পারে। how scripting language run : যেহেতু স্ক্রিপ্টিং ল্যাঙ্গুয়েজে compiled করার প্রয়জন পরে না। তাই ইটা রান করার জন্য ব্যবহার করা হয় the interpreter (often part of the web server, like Apache or Nginx) এরা কোড লাইন বা লাইন পরে। machine instructions এর মতো করে যাতে কম্পিউটার বুজতে পারে। php ভার্সন চেক করার জন্য ব্যবহার করা হয় : phpversion() Web Host : ওয়েব হোস্ট হলো একটা নির্ধারিত জায়গা যেখানে ওয়েবসাইট এর ফাইল রাখা হয়. অনেকটা ভাড়া করা জায়গার মতো। যেখানে যে কেউ তার মনের মত স্পেস ভাড়া করতে পারে তার ওয়েবসাইট এর জন্য। Bluehost, HostGator, and GoDaddy. এই কোম্পানি গুলো এই স্পেস ভাড়া দেয়। Web Server: ওয়েব সার্ভার হলো একটা টুল (tool) যেটা আমার ওয়েবসাইট রান করতে এবং ভিসিটরদের দেখতে সাহায্য করে। like Apache or Nginx server .
