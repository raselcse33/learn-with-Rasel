PHP Introduction scripting language : স্ক্রিপ্টিং ল্যাঙ্গুয়েজে হচ্ছে এক প্রকার প্রোগ্রামিং ল্যাঙ্গুয়েজে যেটা compilation ছাড়াই excute হয়। example : php ,javascript of Non-Scripting Languages : নন স্ক্রিপ্টিং ল্যাঙ্গুয়েজে হচ্ছে Compiled ল্যাঙ্গুয়েজে। মানে এই ল্যাঙ্গুয়েজে টাকে কম্পিউটার এ রান করার পূর্বে এটি কে compiled অর্থাৎ মেশিন ল্যাঙ্গুয়েজে (বাইনারি কোড এ ) রূপান্তর করতে হবে। example : c ,c ++,java compiled : compilied হলো সেই জায়গা যেখানে আমার লেখা কোড টাকে machine language (binary code) convert করা হয়, যাতে কম্পিউটার বুজতে পারে এবং ঠিক ভাবে রান করতে পারে। how scripting language run : যেহেতু স্ক্রিপ্টিং ল্যাঙ্গুয়েজে compiled করার প্রয়জন পরে না। তাই ইটা রান করার জন্য ব্যবহার করা হয় the interpreter (often part of the web server, like Apache or Nginx) এরা কোড লাইন বা লাইন পরে। machine instructions এর মতো করে যাতে কম্পিউটার বুজতে পারে। php ভার্সন চেক করার জন্য ব্যবহার করা হয় : phpversion()
